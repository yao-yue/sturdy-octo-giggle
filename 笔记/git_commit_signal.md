表情代码代表含义
✨:sparkles:引入新功能（Feature）
🐛:bug:修复 Bug
💄:lipstick:更新 UI 或样式文件（CSS/HTML）
📝:memo:编写文档（README、注释等）
🎨:art:改进代码结构 或格式化代码，不影响功能
⚡️:zap:提升性能
♻️:recycle:重构代码
🔒:lock:修复安全漏洞
🚀:rocket:部署上线
🔧:wrench:修改配置文件
➕:heavy_plus_sign:增加依赖
➖:heavy_minus_sign:删除依赖
👷:construction_worker:添加或更新 CI 构建系统
🔨:hammer:开发脚本或工具
